# Eye Data Indexing System (AVL Tree in C)

This project is a eye data indexing system implemented in C using an AVL Tree.

The system loads eye measurement data from a file and stores it in a self-balancing AVL tree to enable efficient searching and indexing of user eye records.

## Features

- AVL Tree based indexing
- Self-balancing rotations (LL, RR, LR, RL)
- Linked list for handling duplicate user names
- File-based data loading
- Search eye information by user name
- Display full index of eye records
- Identify the youngest user based on iris-to-pupil ratio

## Data Structures Used

- AVL Tree
- Linked List
- Dynamic Memory Allocation

## Program Menu

1. Display the full index of eyes
2. Display the details of a user's eyes
3. Display the youngest user
4. Exit

## Files

main.c → main program and menu system  
avltree.c → AVL tree implementation  
avltree.h → AVL tree header definitions
